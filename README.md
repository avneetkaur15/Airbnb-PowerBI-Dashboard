## 📊 Global Airbnb Performance Dashboard

### 📌 Project Overview
An interactive Power BI dashboard designed to analyze Airbnb listings, ratings, reviews, pricing trends, host information, and market performance across major cities worldwide.It is intended for data analysts, business intelligence enthusiasts, tourism researchers, and anyone interested in understanding the global Airbnb marketplace through data-driven insights.

### 🎯 Goal of the Dashboard
- Analyze Airbnb market growth over time.
- Compare performance across major cities.
- Understand review and rating patterns.
- Identify high-performing locations.

### 🛠️ Tech Stack
- 📊 Power BI Desktop – Main data visualization and dashboard development tool.
- 🔄 Power Query – Used for data cleaning, transformation, and preparation.
- 🧮 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, and analytical metrics.
- 🔗 Data Modeling – Relationships established between Listings and Reviews datasets.
- 🎨 Bookmarks – Used to create interactive switching between Overall Rating and Detailed Rating views.

### Data Source
- The dataset used in this project was obtained from the Maven Analytics Data Playground, a platform that provides real-world datasets for data analysis and business intelligence projects.
- The Airbnb dataset contains detailed information about listings, hosts, reviews, ratings, pricing, and property characteristics across multiple global cities. The data was provided in separate listings and reviews tables along with data dictionaries describing the dataset structure and attributes.
